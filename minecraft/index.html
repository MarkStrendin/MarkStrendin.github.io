How to Import this modpack

With the Twitch app, you can have multiple modpacks installed, with seperate configs, and they won't conflict with eachother.
  1. Have the Twitch Desktop App installed on your computer
  2. Download the Zip file linked above, and remember where you saved it
  3. In the Twitch Desktop App, go to Mods, then select Minecraft
  4. Click Create Custom Profile
  5. Click Import link
  6. Give it the zip file that you downloaded in step 2
  7. Wait for it to download the mods
  8. You're done. You may need to adjust your RAM using the instructions below for best performance.


You may need to adjust your Java RAM to avoid crashes or horrible client-side lag

The modpack may crash when starting up, due to insufficient RAM. This just means you need to tell the Twitch launcher to let it use more RAM. It may launch fine, but you get horrible lag every few seconds - this means you need to allocate more memory and relaunch the game.
  1. In the Twitch Desktop App, go to Mods, then select Minecraft
  2. Select the modpack you wish to edit (probably the one you just installed using the instructions above)
  3. Click the Gear Icon in the top-right, next to the "Play" and "Versions" buttons
  4. Select Profile Options
  5. Uncheck Use System Memory Settings
  6. Set the slider to an appropriate number.
    • This depends on your system, and how much RAM you have. I recommend above 5000. Try it and see what works for you, and adjust it if necesary.
